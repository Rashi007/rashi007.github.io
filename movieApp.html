Movie App

#SQLite #movie API #Butterknife #Volley

Overview:

This project was made under Udacity Nanodegree programme . Most of us can relate to kicking back on the couch and enjoying a movie with friends and family. In this project, we build an app to allow users to discover the most popular movies playing or the most top-rated one and create list of their favourite ones. The user could read reviews, play trailers The data was fetched from the Movie database api.
I also made different layout for tablets like in the third image

Features:

  • Persistent storage using content provider with the help of SQLite database to display them even in offline mode.
  • Fetching data from api
  • Responsive design for tablets

Libraries:

  • ButterKnife
  • Volley
